【输出格式】
输出必须是一个 JSON 数组，数组中每个元素对应一道多选题。
每个元素包含以下字段：情境标签、情境、问题、A、B、C、D、E、答案、干扰项设计说明。

示例（仅展示结构，实际内容须按维度要求生成）：
```json
[
  {
    "情境标签": "不超过八个字的情境概括",
    "情境": "150-350字叙事性文本...",
    "问题": "以下哪些选项准确描述了[主体]在[时刻]的心智状态？（多选，选出所有正确项）",
    "A": "关于某方面的描述",
    "B": "关于某方面的描述",
    "C": "关于某方面的描述",
    "D": "关于某方面的描述",
    "E": "干扰项",
    "答案": "ABD",
    "干扰项设计说明": {
      "高混淆干扰项": "与正确项在表述上高度相似，但在归因对象或层次上存在偏差的说明",
      "过度推断干扰项": "超出情境可观测信号范围，将推断结论扩大化的说明",
      "混淆主体干扰项": "将一个主体的状态错误归属于另一主体的说明"
    }
  }
]
```

注意：
- 答案字段填写所有正确选项的字母组合（如"ABD"），不要填写选项文字。
- 每道题是数组中的一个独立元素。
- 只输出 JSON 数组，不要输出任何解释、分析或其他文字。
